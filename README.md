# 🏠 HomeWare Store

Selamat datang di **HomeWare Store**, sebuah website e-commerce sederhana yang menjual berbagai peralatan rumah tangga berkualitas dengan tampilan modern menggunakan **HTML, Tailwind CSS, dan JavaScript**.

---

# 📖 Tentang Project

HomeWare Store dibuat sebagai proyek pembelajaran website e-commerce. Website ini menyediakan berbagai produk rumah tangga dengan desain yang responsif dan mudah digunakan.

---

# 🚀 Cara Menjalankan Project

1. Download atau Clone repository ini.

2. Buka folder project menggunakan Visual Studio Code.

3. Jalankan menggunakan **Live Server**.

4. Website akan menampilkan halaman Welcome kemudian otomatis masuk ke halaman utama.
